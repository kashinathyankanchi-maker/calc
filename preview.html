GitCalc

A premium GitHub-themed Calculator built as an Android Flutter app. It offers a standard calculation interface, a developer search utility, and a timber log CBM calculator.

  • Fully responsive layout in material design
  • GitHub dark-palette aesthetic
  • Math-expressions parsing engine
  • Integrates live profile stats with your equations
  • Dedicated Wood Log Cubic Meter (CBM) Calculator

How to use the preview:

  1. Standard Calculator operates in the first tab.
  2. Search developer statistics in the second tab (Dev Stats) and click metrics to load values.
  3. Calculate log volume in the third tab (CBM Calc) and insert values directly into your equations!
